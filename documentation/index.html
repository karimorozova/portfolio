Tomas_Template folder

the folder consists of:

  • preview.html (theme preview all styles available for the theme)
  • index_1.html (Default Dark version)
  • index_2.html (Particles Dark version)
  • index_3.html (Default Light version)
  • index_4.html (Particles Light version)
  • preview_assets (images and css styles for the preview html page)
  • css folder (theme css files)
  • js folder (theme js files)
  • php folder (theme php files)
  • images folder (theme images used)
  • webfonts folder (theme font used to make icons)

HTML Structure

HTML Head consists of :

  • title
  • meta
  • google fonts (PT Sans)

  • CSS Main Files

HTML Body consists of :

CSS Structure

CSS folder consists of :

  • style_dark.css (main dark version styles file)
  • style_light.css (main light version styles file)
  • bootstrap.min.css (Bootstrap css file)
  • responsive.css (css respnsive layout [moblie layout] file)
  • all.min.css (Font awesome plugin)
  • jquery.animatedheadline.css (Home Head text animation)
  • animate.css (Animation plugin)
  • swiper.css (slider home plugin)
  • fancybox.css (Gallery image magnification plugin)
  • odometer-theme-minimal.css (Counter plugin)

JS Structure

JS folder consists of :

  • main.js (main js file)
  • jQuery.js (jquery js library)
  • jquery.appear.js (from jquery library extentions)
  • bootstrap.bundle.min.js (Bootstrap js file)
  • jquery.animatedheadline.min.js (Home head text animation)
  • swiper.js (slider plugin)
  • odometer.min.js (Counter plugin)
  • wow.min.js (animation on scrolling plugin)
  • fancybox.umd.js (pop up images & videos plugin)
  • mixitup.min.js (shuffling images plugin)
  • particles.min.js (canvas shapes plugin)
  • particles_shape_dark.js (for dark_shapes particles plugin)
  • particles_shape_light.js (for white particles plugin)

PHP Structure

PHP folder consists of :

  • mailer folder (PHP plugin for sending information throught SMTP)
  • composer.json (from PHP plugin files)
  • composer.lock (from PHP plugin files)
  • mail.php (PHP file which container SMTP information about sending informations)
  • contact.php (PHP file for reciveing data from contact form)
                
try {
    $mail->isSMTP();                               //Send using SMTP
    $mail->Host       = 'smtp.gmail.com';          //Set the SMTP server to send through
    $mail->SMTPAuth   = true;                      //Enable SMTP authentication
    $mail->Username   = 'demo@example.com';        //SMTP username 
    $mail->Password   = 'demopw';                  //SMTP password
    $mail->SMTPSecure = "ssl";
    $mail->Port       = 465;

    //Content
    $mail->isHTML(true);
    $mail-> CharSet= "utf-8";
    echo 'Message has been sent'; //message shown on sucsess message send
}

Image Customize

before changing an image you should know important information on images you will replace.
you can replace any image with another and the image will be resized automatically to prevent the change in aspect ratio but some images should have the same aspect ratio to prevent errors

  • Blog section images (images/blog/.. )images should have the same Aspect ratio
  • Porfolio section images (images/projects/.. )prefered to have aspect ratio -> 1:1
  • Reviews section images (images/clients/.. ) Aspect ratio -> 1:1
  • About section image (images/about.jpg) prefered to be Portrait image (height is greater than width)

Home Heading Customize

data-order is the order of images if you want to add another image go to the last order number and increase one to the new image you will put
